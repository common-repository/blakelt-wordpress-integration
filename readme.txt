== Description ==

Blake.lt integration with your Wordpress blog.

== Installation ==

1. Upload directory `/blakelt-wordpress-integration/` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use it as a widget or standalone function in your template files. More at Usage.

== Usage ==
**Widget**

You can use this plugin as a widget. Go to `Appearance > Widgets`

**Standalone function**

More at [Blake.lt Wiki](http://blakelt.pbwiki.com/Wordpress-priedas "Blake.lt Wiki")
